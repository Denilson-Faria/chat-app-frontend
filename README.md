# KonnexApp

Aplicação de chat em tempo real utilizando React, Node.js e Socket.io.

## 🔗 Links

- Frontend: https://github.com/Denilson-Faria/konnex-frontend
- Backend: https://github.com/Denilson-Faria/konnex-backend
- Deploy: https://konnexapp.vercel.app

## 🚀 Tecnologias

### Frontend
- React
- Context API
- Socket.io Client

### Backend
- Node.js
- Express
- MongoDB
- JWT
- Socket.io

## ✨ Funcionalidades

- Autenticação JWT
- Chat em tempo real
- Proteção de rotas
- Persistência de mensagens
- Múltiplos usuários online
